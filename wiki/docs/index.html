おすすめの読み順

  1. まず 環境準備とプラグイン導入 で Python / discord.py / FFmpeg をセットアップ。
  2. はじめての Bot でスラッシュコマンドの最小構成を体験。
  3. UI が必要なら ボタン・モーダル付きフォーム、管理なら モデレーション & ログ を追加。
  4. 音声機能が欲しくなったら 音楽プレイヤー を組み込む。
